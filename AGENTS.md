## Product goal
Build a premium-feeling AI personal aesthetic optimization app focused on progress, routines, and practical recommendations rather than superficial rating.

## UX rules
- Mobile-first
- Premium minimal UI
- One primary action per screen
- Strong spacing and typography
- Clear empty, loading, success, and error states
- Use supportive, non-shaming language
- Avoid clutter

## Code rules
- TypeScript everywhere
- Reusable components
- Clean folder structure
- Avoid unnecessary dependencies
- Keep functions readable and short
- Prefer maintainability over cleverness

## Safety rules
- No harmful appearance scoring language
- No unsafe dieting guidance
- No fake medical certainty
- Always frame skincare/hair/body advice responsibly
- Include informational-not-medical disclaimer where needed

## Workflow
- Make a plan first
- State assumptions before coding
- Build in steps
- After each major step, explain how to test
- Summarize changed files after each major implementation block
